Projects Yeti is a freelancing application where users can browse and view various freelance projects posted by various companies. It provides a platform for the freelancers to find and bid for various projects. Freelancers can manage their profile and showcase their experience and certifications as well as their skills. They can attract companies to award them their listed projects. Users can browse and bid for projects. Companies will evaluate their profiles and bids, including the bidding amount and their CVs, based on these details, freelancers can be awarded the projects and start working on them. After project completion, they receive their payment completing the freelancing platformâ€™s objectives. The mobile application has been built using flutter. Hive database has been used for local storage and a remote API has been built using Node.js with which the application connects.
